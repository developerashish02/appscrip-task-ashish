## Installation

To get started with the project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/developerashish02/appscrip-task-ashish.git
   ```

2. **Install packages**

   ```bash
   npm install
   ```

3. **start server**

   ```bash
   npm run dev
   ```
